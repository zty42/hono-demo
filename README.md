# hono-example-blog

CRUD web API for Blog.

## Features

- Cloudflare Workers
- KV
- TypeScript
- `wrangler` for develop and deploy
- Test with Vitest

## Endpoints

- `GET /posts`
- `POST /posts`
- `GET /posts/:id`
- `PUT /posts/:id`
- `DELETE /posts/:id`

## Usage

Install dependencies:

```sh
yarn install
```

Rename `wrangler.example.toml` to `wrangler.toml`:

```sh
mv wrangler.example.toml wrangler.toml
```
